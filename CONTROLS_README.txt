__Controls__

Escape - Kill application
R - Quick Restart

# MOVEMENT
A / Left Arrow - Move Left
D / Right Arrow - Move Right
W / Up Arrow - Jump
Space - Dash
L_Ctrl - Tilt Camera


# INTERFACE
P - Pause Menu
Enter -  Choose Option


# Cheats / Developer tools
1 - Switch to Baby Character
2 - Switch to Adult Character
3 - Switch to Old Character

8 - Jump to Baby Zone
9 - Jump to Adult Zone
0 - Jump to Old Zone
